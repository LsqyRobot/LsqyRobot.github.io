小工具使用说明

windows 软件推荐

  • Everything (搜索文件工具)
  • snipaste (贴图 + 截屏)
  • autohotkey (自定义快捷键,有编程底子的很快就能上手)
  • focusky (相对简单的ppt制作软件,且花里胡哨)
  • Typora (好用的markdown编辑器)
  • Anaconda (开源包管理器)
  • winedt (latex编辑器)
  • texlive (miktex有时真的不给力，还不如安装texlive可以解决一些不必要的麻烦)
  • Evernote (印象笔记，可以方便的管理自己的笔记)
  • visio (可以绘制出好看到变态的软件)
  • vs (开箱即用的编辑器，相对于vim，对新手非常友好,不过代价就是让人变懒了,类似的还有sublime，atom等等非常优秀的编辑器,不好意思，我用vim)
  • Mendeley (目前为止发现最适合自己的文献管理工具，目前为止放了近一千本书籍进去(chi hui)了，而且在windows可以同步，不必担心这些书籍找不到)
  • Teamviewer (非常好用的远程控制工具)

mac 软件推荐(win上有的就不重复推荐了)

  • iText(一款不错的ocr识别工具，不过需要收费)
  • iMovie (非常强大的视频剪辑软件)
  • docker (非常强大且好用的虚拟机)
  • keynote (非常有水平的ppt演示工具)

数学软件推荐

  • Mathematica (绝对的神器,无论是数值计算还是符号处理等等等等，都是一等一的水平)
  • Matlab (工科神器)
  • Lingo (运筹优化)
