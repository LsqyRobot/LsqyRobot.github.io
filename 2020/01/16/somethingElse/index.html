小工具使用说明

将一些小工具的使用放在一个文件夹中， 有时候突然忘了，来查阅一下，不然找个功能都要花很长时间，多不值得。

绘图工具

plantUML

使用HTML进行格式化(Formatting using HTML)，可以使用少量的HTML标签来格式化文本, 该内容请点击链接, 由于这个脚本可以绘制很多种图，免费开源，绘制的图形虽然不如专门的工具，比如visio, 但是对于自己快速绘制流程图等等，是很有帮助的。

windows 软件推荐

  • Everything (搜索文件工具)
  • snipaste (贴图 + 截屏)
  • autohotkey (自定义快捷键,有编程底子的很快就能上手)
  • focusky (相对简单的ppt制作软件)
  • Typora (好用的markdown编辑器)
  • Anaconda (开源包管理器)
  • winedt (latex编辑器)

由于几年没怎么使用win了，所以记得的软件不多。

Hexo 上的一点小技巧

插入固定图片大小时可用如下指令

<img src="/images/your.png" width="70%" height="50%">
