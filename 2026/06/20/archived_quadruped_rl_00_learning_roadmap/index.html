四足机器人强化学习系列 00：从零开始的学习路线图
目录

为什么要单独学“四足机器人强化学习”

如果只是泛泛地学强化学习，通常会先从 GridWorld、CartPole、Atari、MuJoCo HalfCheetah 讲起。这些例子很好，但和四足机器人真正落地之间还隔着几层：机器人动力学、关节控制、仿真器、奖励函数、接触、地形、域随机化、实机安全。

所以这个系列不追求“把所有 RL 理论讲完”，而是围绕一个目标：看懂并能复现四足机器人运动控制里的强化学习训练流程。

一句话路线

先建立 PPO 与 MDP 的最小直觉，再理解四足机器人的状态/动作/奖励，随后用仿真并行训练，最后学习 Sim2Real 与实机安全验证。

这个系列会怎么写

计划按下面顺序写：

  1. 00 路线图：知道要学什么、先后顺序是什么。
  2. 01 PPO 与 MDP：四足机器人训练里最常见的算法骨架。
  3. 02 仿真训练流程：Isaac Gym / legged_gym / rsl_rl 这类工程是怎么跑起来的。
  4. 03 奖励函数设计：为什么机器人会学会走路，奖励到底在约束什么。
  5. 04 Sim2Real：仿真策略怎么尽量安全地迁移到真机。
  6. 05 复现实验清单：给自己搭一个从仿真到日志分析的练习路径。

后面可以继续追加：地形课程学习、速度命令跟踪、跳跃/攀爬、轮足机器人、模仿学习、世界模型、视觉 RL 等。

你需要具备什么基础

最低门槛：

  • 会一点 Python。
  • 知道机器人有“关节角、关节速度、IMU、足端接触”这些量。
  • 能接受一开始不追求公式全懂，先跑通工程闭环。

更理想的基础：

  • Linux / Conda / Git 基础。
  • PyTorch 基础。
  • ROS2 基础。
  • 一点动力学概念：质心、姿态、关节力矩、接触力。

四足 RL 的核心闭环

可以把训练过程理解成下面这个循环：

四足机器人强化学习真正难的地方，不只是 PPO，而是：

  • obs 里放什么？
  • action 表示什么？位置目标、力矩、还是残差？
  • reward 怎么设计才不会学出奇怪动作？
  • reset 条件怎么写？
  • 仿真和真机差异怎么处理？
  • 学出来之后怎么安全上真机？

建议学习节奏

如果你是零基础，我建议这样学：

第一阶段：只求建立直觉

目标：不用完全推公式，但要知道每个词在四足机器人里对应什么。

需要掌握：

  • state / observation：机器人“看到”的信息。
  • action：策略网络控制机器人做什么。
  • reward：希望机器人更像什么。
  • episode：一次从开始到摔倒/结束的尝试。
  • policy：从观测到动作的函数。
  • value：当前状态未来大概能拿多少分。

第二阶段：跑通一个现成工程

推荐优先看：

  • legged_gym：经典四足 PPO locomotion 工程。
  • rsl_rl：常与 legged_gym 配套的 PPO 实现。
  • Isaac Gym / Isaac Sim / Isaac Lab：NVIDIA 生态下的并行仿真训练工具。

这一阶段的目标不是改算法，而是能回答：

  • 训练入口在哪？
  • 环境类在哪里？
  • reward 在哪里？
  • obs 在哪里拼接？
  • action 怎么映射到 PD target？
  • 日志怎么看？

第三阶段：改奖励、看现象

这一步最关键。比如：

  • 增大速度跟踪奖励，机器人更努力追命令速度。
  • 增大力矩惩罚，动作更省力，但可能变慢。
  • 增大脚滑惩罚，步态更稳，但可能保守。
  • 调整 base height 奖励，身体高度和姿态会变。

你会发现，四足 RL 很像“用奖励塑造行为”。奖励写得不好，模型可能会钻空子。

第四阶段：理解 Sim2Real

仿真里能走，不代表真机能走。真机问题包括：

  • 电机延迟。
  • 摩擦系数不同。
  • 关节零位误差。
  • IMU 噪声。
  • 地面材质不同。
  • 电池电压变化。
  • URDF / 惯量参数不准。

因此需要：

  • 域随机化。
  • 观测噪声。
  • 动作延迟建模。
  • 推力/力矩限制。
  • 安全保护和逐级测试。

  • legged_gym: https://github.com/leggedrobotics/legged_gym
  • rsl_rl: https://github.com/leggedrobotics/rsl_rl
  • Isaac Gym: https://developer.nvidia.com/isaac-gym

最后

这个系列的重点不是堆概念，而是帮你形成一个工程视角：四足强化学习 = 策略网络 + 仿真环境 + 奖励函数 + 大量并行采样 + Sim2Real 安全约束。
