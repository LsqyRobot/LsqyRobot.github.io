前言

对自己曾今遇到的错误,而且以后有极大概率会遇到的错误进行内容的收集并引用出处；

Latex 报错：Package fontspec Error: The font “KaiTi” cannot be found

极大概率是由于使用了 newtx, 将相关内容在.cls文件中注释掉即可；此问题的答案在知乎的回复中找到的；

latex文件里面一个章节放的图表太多编译出现too many unprocessed floats的解决方案

此问题在CSDN里面找到的答案, 在第18个图或表后添加 \clearpage即可
